# Команда signup

### Описание
Команда используется для регистрации пользователя в системе.
	
### Синтаксис
```
signup <логин> <пароль>
```

### Примечания
Требования к логину:  
От 5 до 32 латинских букв или цифр.  

Требования к паролю:  
От 7 до 32 любых символов, содержащих один символ из:  
\.,;!\?&\^@#\$%\*\(\)\[\]{}  

### Пример
```
signup user123 password.
```
